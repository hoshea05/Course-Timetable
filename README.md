## Description
A simple timetable system built with Node.js core modules and MySQL.

## Solution
- Uses `http`, `fs`, `url` modules for routing
- Dynamically generates HTML templates
- Performs CRUD operations on MySQL database

## Test Cases
1. Visit `/timetable` → Course list appears
2. Click Add → appears in `/my-schedule`
3. Remove → disappears from my schedule
